### Discover Mountain Resorts

A website that allows users to share some of their favorite parts about a ski resort.  Whether it be a restaurant, a gorgeous view, or even a ski trail.  

## Technologies Used

#Front End
- React
- Redux
- TypeScript
- Webpack
- Bootstrap
- jQuery

#Back End
- Azure
- Entity Framework
- .NET Core
- SQL Server
- Identity 2.0
